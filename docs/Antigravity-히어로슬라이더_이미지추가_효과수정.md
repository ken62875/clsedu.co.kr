# 메인 히어로 섹션 슬라이더 업데이트

## 과제 목표
* 히어로 섹션 슬라이더(`HeroSlider.tsx`) 목록 맨 처음에 사용자 제공 이미지 추가.
* 슬라이더가 줌인/줌아웃 되며 전환되는 애니메이션 효과(Scale)의 깊이 및 지속시간 연장.

## 구현 상세 내역

### 1. 신규 슬라이드 이미지 추가
* 기존 `images` 배열 맨 첫 번째 위치에 요청하신 이미지 URL(`https://media.clsedu.co.kr/KakaoTalk_Photo_2026-03-31-21-41-01%20001.jpeg`)을 삽입하였습니다.
* 이로써 전체 슬라이드는 총 4장으로 롤링됩니다.

### 2. 줌 효과 깊이 개선 (Scale & Transition)
* 줌 인/아웃 느낌을 더 부각하기(깊이를 길게) 위해 `transform: scale` 수치를 기존 `1.08`에서 **`1.15`** 로 상향 조정했습니다.
* 스케일링이 너무 급격하게 일어나는 것을 방지하고 끊임없이 깊게 들어가는 느낌을 주고자 CSS `transition` 중 `transform` 변화 지속 시간을 기존 `7s linear`에서 **`10s linear`**로 연장했습니다.

## 확인 및 검증 방법
1. 로컬 환경(`http://localhost:3000`) 접속 시 최상단 메인 히어로 섹션 확인.
2. 접속 즉시 방금 추가된 사진이 첫 슬라이드로 띄워지는 것을 확인.
3. 5초 간격으로 사진이 교체되는 동안, 부드럽고 더 깊숙하게(scale 1.15) 줌-인 효과가 점진적으로 일어나는지 육안으로 확인.
