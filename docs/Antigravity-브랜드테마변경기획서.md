# CLS에듀케이션 브랜드 색상 및 서브페이지 리뉴얼 기획서

## 1. 개요
*   **목적**: 기존 네이비/골드 톤으로 작업된 웹사이트를 최종 확정된 공식 로고 가이드(final_clseducation_logo.pdf)에 맞추어 **오렌지와 블랙** 메인 컬러 시스템으로 전면 개편합니다.
*   **추가 작업**: 대명학원, 정상어학원 등 1군 교육기관의 레이아웃을 벤치마킹하여 **강사 소개** 화면과 **프로그램 소개** 화면을 단순히 비워두는 것이 아닌, 고품질의 실제 서비스 수준 목업(Mockup) UI로 구축합니다.

## 2. 세부 개발 내용

### A. 디자인 시스템 & 로고 업데이트
*   `globals.css` 테마 변수 변경 (Navy -> Black, Gold -> Orange).
*   tailwind 클래스명 전역 검색 및 치환 (`cls-navy` -> `cls-black`, `cls-gold` -> `cls-orange`).
*   `public/logo-clsedu-landscape.webp` 이미지를 로드하여 `Header`와 `Footer` 로고 영역을 대체.

### B. 프리미엄 강사 소개 (Teachers)
*   **디자인 컨셉**: 인물 사진 중심의 세련된 프로필 그리드 레이아웃.
*   **구성 요소**: 원장님 및 과목별 대표 강사 프로필, 출신 대학 및 대치동/목동 등 주요 경력사항, 교육 철학 명언 박스.

### C. 프리미엄 프로그램 소개 (Program)
*   **디자인 컨셉**: 체계적이고 과학적인 커리큘럼을 보여주는 인포그래픽/다이어그램 스타일.
*   **구성 요소**: 
    - 대상별 코스(초/중/고)
    - 단계별 학습 프로세스 (예시: 1단계 개념수립 -> 2단계 심화응용 -> 3단계 1:1 클리닉)
    - 내신 및 입시 윈터스쿨 시스템 안내 박스.

## 3. 진행 방법
*   기획 내용의 승인이 떨어지는 즉시 `app` 디렉토리 내의 각 컴포넌트와 CSS 파일을 수정하고, `task.md`를 통해 진척도를 추적합니다. 끝나면 Github 커밋 명령어 제공.
