# CLS에듀케이션 웹사이트 구축 기획서

## 1. 프로젝트 개요
*   **목적**: CLS에듀케이션 학원의 전문성과 교육 철학을 온라인에 효과적으로 전달하고, 학부모 및 학생들의 상담 문의(Lead Generation)를 유도하는 현대적이고 신뢰감 있는 랜딩페이지 구축.
*   **타겟 고객**: 초등, 중등, 고등학생을 자녀로 둔 학부모 및 학생 (내신 및 입시 위주)
*   **주요 컨셉**:
    *   **슬로건**: "진짜 실력은 속도가 아니라 깊이에서 나옵니다."
    *   **핵심 철학**: "최고의 가치는 사람입니다."
    *   **디자인 톤앤매너**: 네이비(신뢰, 전문성)와 골드/오렌지(포인트, 강조)의 조합. (참조: mockup.ivynet.co.kr, gyopum.com)

## 2. 기술 스택
*   **Framework**: Next.js (최신 버전, App Router 사용)
*   **언어**: TypeScript
*   **스타일링**: Tailwind CSS (사용자 승인 시) 또는 Vanilla CSS Modules
*   **기타**: Responsive Design (반응형 웹), SEO 최적화

## 3. 주요 메뉴 및 페이지 구조

1.  **메인 (Main 랜딩)**
    *   히어로(Hero) 영역: 학원 전경/공부 모습 배경 이미지 + 핵심 슬로건
    *   교육 철학 요약: 1:1 맞춤형 로드맵, 투명한 피드백 강조
    *   프로그램 요약 슬라이더: 초/중/고 입시 프로그램
    *   Call To Action (CTA): 무료 상담 신청 폼/버튼

2.  **학원소개 (About)**
    *   CLS에듀케이션의 약속 및 핵심 가치
    *   학원의 특별함 (공부 근육 역량 강화)
    *   학부모님 후기 인용구
    *   오시는 길 (지도 API 및 주소 안내)

3.  **강사소개 / 교과과정 / 프로그램 / 스토리 (블로그)**
    *   우선 임시 데이터(Mockup)로 퍼블리싱 (콘텐츠 수급에 따라 정식 내용으로 교체 예정)

4.  **문의 (Contact)**
    *   빠른 상담 신청을 위한 폼 화면, 및 Floating (우측 하단) 고정 상담 버튼

## 4. 진행 계획
1.  **Phase 1 (기획 및 승인)**: 본 기획서 및 Next.js 초기화, Tailwind CSS 사용에 대한 사용자 승인 과정 (현재 대기)
2.  **Phase 2 (프로젝트 초기화)**: `clsedu.co.kr` 리포지토리에 Next.js(App Router) 스캐폴딩 설정. 글로벌 CSS 톤 세팅.
3.  **Phase 3 (페이지 & 컴포넌트 개발)**: 
    *   공통 레이아웃 (Header, Footer, Floating CTA 버튼) 구축.
    *   메인 랜딩페이지 스크롤 애니메이션 및 섹션별 조립.
    *   서브 페이지(학원소개 등) 정보 구조화 및 디자인.
4.  **Phase 4 (검증 및 배포 준비)**: 모바일 반응형 최종 테스트 후 Github 커밋 및 Coolify를 통한 실제 배포 진행.
